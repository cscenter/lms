  1. Памятка по форматированию текста

Для оформления текста на сайте повсеместно используется Markdown, а также Latex для формул.
Ниже небольшой пример, демонстрирующий их возможности.

  • Что пишем
  • Что получаем 
# Шпаргалка по Markdown

В поле ввода поддерживается формат Markdown и $\LaTeX$.

**жирный шрифт**

*курсив*

`моноширинный шрифт`


Для разбиения на параграфы используется пустая строка:

Параграф 1.
Продолжение первого параграфа.

Параграф 2.

Поддерживаются списки

  * Один.
  * Два.
    Продолжение пункта два.
  * Три.

Обратите внимание на пустую строку до и после списка.

Можно использовать заголовки разных уровней:

## Второй уровень

### Третий уровень


Для вставки кода можно использовать конструкцию следующего вида:

```
# Useful comment
foo = 42
for i in range(foo):
    print("I <3 Computer Science")
```

Обратите внимание на пустую строку до символов обратного апострофа.

То же самое, но с использованием 4 пробелов:

    # Useful comment
    foo = 42
    for i in range(foo):
        print("I <3 Computer Science")


Формулы окружаются долларами $F_n$.
Чтобы формула была на отдельной строке, её следует окружить двумя парами
долларов: $$F_n = F_{n-1} + F_{n-2}$$

# Шпаргалка по $\LaTeX$

Индексы: $x_i, x^j, x_i^j, x_{i+1}$.

Некоторые функции и символы: $\pi, e, \sin{x}, \log(\mathbf{V} + \mathbf{E})$.

Дроби: $\frac{1}{2}, \binom{6}{4}$.

Суммы: $\sum_{i=0}^n i^2$.

Произведение $f$, $o$ и $o$: $foo$.

Переменная `foo`: $\mathtt{foo}$.
